###### before start:
```
1) open your .gitignore file
~$ code .gitignore

2) check it to have at least the next lines:
dist/
src/styles/
node_modules/

3) install node modules, check if everything works fine
~$ npm install     
~$ npm start

if you see opened index.html and sass watching for changes, all is cool
```
